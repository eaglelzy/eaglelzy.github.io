rn跳转调研

简介

rn的跳转分为两部分:

  1. rn页面之间的跳转
  2. rn和native互跳

rn页面之间的跳转

rn 页面之间的跳转使用的是react-navigation,该库主要包含3个跳转相关组件:

  1. StackNavigator(页面之间的跳转)
  2. TabNavigator(Tab之间的跳转)
  3. DrawerNavigator(android侧滑抽屉的跳转)

使用react-navigation必须要安装react-navigation包:

rn和native互跳

native跳rn

这种跳转跟本地的跳转是一样的, 直接用startActivity或startActivityForResutl方式打开rn的承载页即可.
需要注意的是可以通过重写getLaunchOptions()来向rn端传递参数.

rn跳native

rn跳native可以通过扩展module来实现, 需要设计一套路由协议, 具体参考src/bridge/BridgeSample.js.

相关源码在src/navigate/目录下.
更多示例可以参考: github

react-navigation跳转时有两个header
